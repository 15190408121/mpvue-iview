## 简介
本项目为`mpvue`引入`小程序原生组件`示例程序，通过`webpack-plugin`的方式将`iview`文件夹下组件引用到`dist`，插件地址：[`import-weapp-component`](https://github.com/JJJYY/import-weapp-component)
